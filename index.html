파일 구조 규칙

프로그램은 엑셀 파일(.xlsx) 내의 Raw 시트명으로 회차를 자동 인식합니다.
시트명은 반드시 아래 형식을 따라야 합니다.

Raw_0K  →  0K 회차
Raw_5K  →  5K 회차
Raw_10K → 10K 회차
Raw_20K → 20K 회차
Raw_30K → 30K 회차
⚠ 대소문자 구분 없음 (Raw_0k, RAW_0K 모두 인식)
Raw 시트 컬럼 구성

Raw 시트의 1행은 반드시 아래 헤더명을 포함해야 합니다.

컬럼명설명필수
호기장비 번호 (1~6)✅
축측정 축 (X 또는 Y)✅
케이스측정 조건 (case1 / case2 / case3)✅
A(μm)Accuracy 측정값✅
R(μm)Precision 측정값✅
B_bar(μm)Hysteresis 측정값✅
허용오차초과(개)Excess Nodes 측정값✅
파일 업로드 방법
1화면 중앙의 업로드 영역을 클릭하거나, 엑셀 파일을 드래그 앤 드롭
2여러 파일을 동시에 선택할 수 있습니다 (각 회차 파일을 한 번에 업로드)
3업로드 후 상단에 인식된 회차 목록이 표시됩니다 (예: 2개 회차 · 0K / 5K)
💡 파일명은 자유롭게 지정해도 됩니다. 시트명으로만 회차를 인식합니다.
모드 선택
모드그래프 라인선택 항목
케이스 기준case1 / case2 / case3 (3개 라인)지표 선택 (A / P / H / EN)
스펙 기준A / P / H / EN (4개 라인)케이스 선택 (case1~3)
💡 스펙 기준에서 A/P/H는 좌측 Y축(μm), EN은 우측 Y축(개)으로 표시됩니다.
12패널 그리드

6호기 × 2축(X/Y) = 12개 패널이 동시에 표시됩니다.
모든 패널은 동일한 Y축 스케일을 사용하므로 호기 간 절대값 비교가 가능합니다.

!실선 점: 실제 측정 회차 / 점선 영역: 미래 예정 회차 (데이터 없음)
패널 하단 열화표

각 패널 아래에 0K 기준 변화율(%)이 자동 계산됩니다.

색상범위의미
초록≤ 105%정상 (0K 대비 5% 이내)
노랑106~115%주의 (5~15% 증가)
주황116~130%경고 (15~30% 증가)
빨강> 130%위험 (30% 초과)
상세보기 팝업

패널을 클릭하면 상세보기 팝업이 열립니다.

1큰 차트에 각 측정값의 Range 밴드가 함께 표시됩니다 (옅은 색 영역 = ±range/2)
2하단 표에 실제 측정값과 ±range/2가 작은 글씨로 함께 표시됩니다
3구분선 아래에 0K 기준 변화율(%) 또는 Δ값이 색상과 함께 표시됩니다
💡 ESC 키 또는 배경 클릭으로 팝업을 닫을 수 있습니다.
호기 비교 탭 사용법
1축 선택: X축 또는 Y축 중 하나를 선택합니다
2기준 선택: 케이스 기준 또는 스펙 기준을 선택합니다
3지표/케이스 선택: 기준에 따라 비교할 지표 또는 케이스를 선택합니다
4비교 대상 선택: 호기 A와 호기 B를 각각 드롭다운에서 선택합니다
💡 두 패널은 동일한 Y축 스케일을 공유하므로 직접 비교가 가능합니다.
비교 차트 구성
모드라인 구성
케이스 기준선택 지표의 case1 / case2 / case3 라인 (3개)
스펙 기준선택 케이스의 A / P / H (좌축) + EN (우축) (4개)
ISO 230-2 측정 지표
표기전체명단위설명
A Accuracy (정확도)μm 목표 위치에 대한 실제 도달 위치의 편차 범위. 값이 작을수록 우수.
P Precision (반복 정밀도)μm 동일 위치를 반복 이동할 때의 산포 범위. 값이 작을수록 우수.
H Hysteresis (히스테리시스)μm 정방향/역방향 이동 시 위치 차이. 절대값으로 표시. 값이 작을수록 우수.
EN Excess Nodes (허용오차 초과)개 허용 오차 범위를 초과한 측정 노드 수. 0에 가까울수록 우수.
케이스(case) 의미
케이스설명
case1측정 조건 1 (첫 번째 측정 세트)
case2측정 조건 2 (두 번째 측정 세트)
case3측정 조건 3 (세 번째 측정 세트)
💡 각 케이스는 동일 호기에서 서로 다른 조건 또는 반복 측정을 의미합니다.
Range (범위)

각 케이스에서 3회 반복 측정한 값의 최대값 − 최소값입니다.
상세보기 팝업의 그래프에서 옅은 밴드로, 표에서 ±range/2로 표시됩니다.
Range가 클수록 측정 재현성이 낮음을 의미합니다.

열화표 색상 기준 (0K 대비 변화율)
색상조건의미
초록≤ 105%정상 — 0K 대비 5% 이내 변화
노랑106% ~ 115%주의 — 5~15% 증가
주황116% ~ 130%경고 — 15~30% 증가
빨강> 130%위험 — 30% 초과 증가
⚠ H(Hysteresis)와 EN(Excess Nodes)은 % 대신 Δ값으로 표시됩니다.
Δ > 0 이면 악화, Δ ≤ 0 이면 개선으로 판단합니다.
